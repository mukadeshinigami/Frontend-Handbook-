## Пример кода

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Демо основ CSS</title>
  <style>
    /* Селектор по тегу — выбирает все параграфы */
    p {
      color: #333;
      line-height: 1.6;
    }
    
    /* Селектор по классу — переиспользуемый стиль */
    .highlight {
      background: yellow;
      padding: 0.25rem;
    }
    
    /* Каскад: более поздние правила переопределяют ранние */
    p {
      color: #555; /* Это правило побеждает */
    }
  </style>
</head>
<body>
  <p>Обычный параграф с унаследованным line-height.</p>
  <p class="highlight">Выделенный параграф.</p>
</body>
</html>
```

## Разбор (80% нового)

### Структура синтаксиса

```css
селектор {
  свойство: значение;
}
```

- **Селектор** (`p`): выбирает элементы
- **Свойство** (`color`): что стилизуем
- **Значение** (`#333`): как стилизуем
- **Объявление**: свойство + значение + точка с запятой

### Базовые селекторы

1. **Селектор по тегу** (`p`): выбирает все элементы `<p>`
2. **Селектор по классу** (`.highlight`): выбирает элементы с `class="highlight"`
3. **Селектор по ID** (`#main`): выбирает элемент с `id="main"` (не показан)

### Свойства и значения

```css
p {
  color: #333;        /* Цвет текста (hex) */
  line-height: 1.6;   /* Межстрочный интервал */
}
```

Распространённые типы значений:
- Цвета: `#333`, `rgb(0,0,0)`, `blue`
- Длины: `1rem`, `16px`, `50%`
- Числа: `1.6` (без единиц для line-height)

### Каскад

Когда несколько правил применяются к одному элементу, более поздние переопределяют ранние:

```css
p { color: #333; }  /* Применяется первым */
p { color: #555; }  /* Переопределяет предыдущее */
```

Результат: параграфы имеют цвет `#555`.

### Наследование

Дочерние элементы автоматически наследуют некоторые свойства родителей:

```css
p {
  line-height: 1.6;  /* Наследуется вложенными span */
}
```

**Наследуются:** `color`, `font-family`, `line-height`  
**Не наследуются:** `padding`, `background`, `border`

## Ключевые паттерны

1. **Внешняя таблица стилей** (для продакшена):
```html
<link rel="stylesheet" href="styles.css">
```

2. **Внутренняя таблица стилей** (для демо/прототипов):
```html
<style>
  /* CSS здесь */
</style>
```

3. **Стилизация через классы** (поддерживаемо):
```css
.btn { /* переиспользуемый стиль кнопки */ }
```

### 3. Внешний CSS (External) — Рекомендуется
Отдельный файл `.css`, подключаемый через `<link>`.

```html
<head>
  <link rel="stylesheet" href="styles.css">

```

**Файл: styles.css**
```css
p { color: blue; }
```

**Применение:** Продакшн, масштабируемость, кеширование.

## Синтаксис CSS

```css
селектор {
  свойство: значение;
  свойство: значение;
}
```

**Пример:**
```css
h1 {
  color: #333;
  font-size: 2rem;
  margin-bottom: 1rem;
}
```

**Компоненты:**
- **Селектор** (`h1`): указывает HTML-элементы
- **Свойство** (`color`): аспект для стилизации
- **Значение** (`#333`): настройка свойства
- **Объявление** (`color: #333;`): свойство + значение + точка с запятой

## Каскад (Cascade)

Несколько правил могут нацеливаться на один элемент. CSS применяет их на основе:

1. **Специфичности** (более конкретные селекторы побеждают)
2. **Порядка объявления** (более поздние правила переопределяют ранние)
3. **Важности** (флаг `!important`)

**Пример:**
```css
p { color: blue; }      /* Применяется первым */
p { color: green; }     /* Переопределяет предыдущее */
```

Результат: параграфы зелёные.

---

## Наследование (Inheritance)

Некоторые свойства автоматически передаются от родительских элементов к дочерним.

**Наследуемые свойства:** `color`, `font-family`, `line-height`, `text-align`  
**Не наследуются:** `margin`, `padding`, `border`, `background`

**Пример:**
```html
<div style="color: red;">
  <p>Этот текст красный (унаследован)</p>
</div>
```

Принудительное наследование:
```css
button {
  font-family: inherit; /* Наследует от родителя */
}
```

## Краткая справка

| Концепция | Резюме |
|-----------|--------|
| **Назначение** | Отделение представления от содержимого |
| **Подключение** | Inline, internal, external (предпочтителен external) |
| **Синтаксис** | `селектор { свойство: значение; }` |
| **Каскад** | Специфичность + порядок определяют применяемые стили |
| **Наследование** | Некоторые свойства передаются потомкам |
| **Комментарии** | `/* комментарий */` |
