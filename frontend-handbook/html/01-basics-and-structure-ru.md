# HTML Основы и Структура

## Введение

HTML (HyperText Markup Language) — это язык разметки, используемый для создания структуры веб-страниц. HTML используется вместе с CSS (для стилизации) и JavaScript (для интерактивности).

---

## Базовая структура HTML документа

Каждый HTML документ должен содержать определённую структуру:

```html
<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Название страницы</title>
  </head>
  <body>
    <h1>Добро пожаловать!</h1>
    <p>Содержимое страницы</p>
  </body>
</html>
```

### Разбор элементов:

| Элемент | Описание |
|---------|---------|
| `<!DOCTYPE html>` | Объявление типа документа (HTML5) |
| `<html>` | Корневой элемент, содержит весь контент |
| `<head>` | Метаинформация о документе (не видна пользователям) |
| `<body>` | Видимый контент страницы |

---

## Элемент `<head>`

Этот раздел содержит метаинформацию о странице.

### Важные элементы `<head>`:

#### 1. `<meta charset="UTF-8">`
Указывает кодировку символов (обязателен для правильного отображения текста):
```html
<meta charset="UTF-8">
```

#### 2. `<meta name="viewport">`
Управляет масштабированием на мобильных устройствах (важна для адаптивного дизайна):
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

#### 3. `<title>`
Название страницы, отображается в вкладке браузера:
```html
<title>Моя первая веб-страница</title>
```

#### 4. `<meta name="description">`
Краткое описание страницы для поисковых систем:
```html
<meta name="description" content="Это описание моей страницы для SEO">
```

#### 5. `<link>`
Подключение внешних ресурсов (CSS, шрифты, иконки):
```html
<link rel="stylesheet" href="styles.css">
<link rel="icon" href="favicon.ico">
```

#### 6. `<style>`
Встроенные CSS стили:
```html
<style>
  body { background-color: #f0f0f0; }
</style>
```

#### 7. `<script>`
Подключение JavaScript:
```html
<script src="script.js"></script>
```

## Элемент `<body>`

Содержит весь видимый контент страницы.

### Основные элементы body:

#### Заголовки
```html
<h1>Главный заголовок (самый важный)</h1>
<h2>Подзаголовок</h2>
<h3>Под-подзаголовок</h3>
<!-- ... до <h6> -->
```

**Лучшие практики:**
- Только один `<h1>` на странице
- Используйте заголовки по порядку (не пропускайте уровни)
- Используйте заголовки для структурирования, а не для стилизации

#### Параграфы
```html
<p>Это обычный абзац текста.</p>
<p>Это ещё один абзац.</p>
```

#### Разрывы строк и горизонтальные линии
```html
<p>Первая строка<br>Вторая строка</p>
<hr> <!-- Горизонтальная линия -->
```

#### Форматирование текста
```html
<strong>Важный текст (жирный)</strong>
<em>Выделенный текст (курсив)</em>
<small>Мелкий текст</small>
<mark>Выделенный текст (фон)</mark>
<del>Удалённый текст</del>
<ins>Вставленный текст</ins>
<sub>Подстрочный индекс</sub>
<sup>Надстрочный индекс</sup>
```

## Практический пример

```html
<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Моя первая веб-страница">
    <title>Hello World - Моя первая страница</title>
  </head>
  <body>
    <h1>Добро пожаловать на мой сайт!</h1>
    <p>
      Это моя <strong>первая веб-страница</strong>. 
      Я использую HTML для создания структуры.
    </p>
    
    <h2>О странице</h2>
    <p>
      Эта страница демонстрирует <em>основные концепции</em> HTML.
    </p>
    
    <hr>
    
    <h2>Информация</h2>
    <p>
      Вода кипит при 100°C<sup>*</sup> на уровне моря.
    </p>
    <small>*При стандартном атмосферном давлении</small>
  </body>
</html>
```

## Основные атрибуты

Атрибуты предоставляют дополнительную информацию об элементах:

```html
<!-- id: уникальный идентификатор -->
<h1 id="main-title">Заголовок</h1>

<!-- class: для стилизации и группировки -->
<p class="intro">Вводный текст</p>

<!-- style: встроенные стили -->
<p style="color: blue;">Синий текст</p>

<!-- title: подсказка при наведении -->
<p title="Это подсказка">Наведите на меня</p>

<!-- data-*: пользовательские данные -->
<div data-user-id="123">Содержимое пользователя</div>
```

## Семантический HTML

Используйте семантические элементы для улучшения доступности и SEO:

```html
<header>
  <h1>Заголовок сайта</h1>
  <nav>Навигация</nav>
</header>

<main>
  <article>
    <h2>Заголовок статьи</h2>
    <p>Содержимое статьи</p>
  </article>
  
  <aside>
    <h3>Боковая панель</h3>
    <p>Дополнительная информация</p>
  </aside>
</main>

<footer>
  <p>&copy; 2025 Мой сайт</p>
</footer>
```

## Валидация HTML

Для проверки корректности HTML используйте:
- **[W3C Validator](https://validator.w3.org/)** — официальный валидатор
- **IDE расширения** — автоматическая проверка в редакторе
- **DevTools браузера** — проверка через консоль
