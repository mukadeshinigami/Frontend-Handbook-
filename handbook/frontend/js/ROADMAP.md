## Продвинутые блоки (следующий этап)

### Блок 12: Тестирование JavaScript
**Файлы:** `03-event-delegation.md` + `03-event-delegation-practice.md`

**Цель:** Понять, как события распространяются через DOM-дерево и как эффективно обрабатывать много элементов.

**Темы:**
- Всплытие событий (event bubbling)
- Погружение событий (event capturing) — кратко
- Делегирование событий (event delegation) — обработка множества элементов через одного родителя
- `event.target` vs `event.currentTarget`
- Практические примеры: динамический список задач (todo-list), таблица с удалением строк

**Практика:**
- Создать динамический список с добавлением/удалением элементов
- Использовать делегирование для обработки кликов на неизвестное количество элементов
