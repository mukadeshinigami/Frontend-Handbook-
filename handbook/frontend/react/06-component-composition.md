
# Блок 6 — Компоненты: композиция и переиспользование

Цель
---
Научиться проектировать интерфейс как набор маленьких, переиспользуемых компонентов и использовать композицию вместо наследования.

Темы
---
- Композиция вместо наследования — передавайте `children` и props.
- `props.children` — как создавать обёртки и layout-компоненты.
- Специализация: создавать базовый компонент и вариации (например, Button с разными стилями).
- Короткое введение в HOC и render props — исторические паттерны, которые часто заменяются hooks.
- Single Responsibility — один компонент = одна ответственность.

Пример
---

```jsx
function Card({ children }) {
	return <div className="card">{children}</div>;
}

function ProductCard({ title, price }) {
	return (
		<Card>
			<h3>{title}</h3>
			<p>${price}</p>
		</Card>
	);
}
```
