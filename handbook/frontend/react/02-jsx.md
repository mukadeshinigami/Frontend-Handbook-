
# Блок 2 — JSX и выражения

Цель
---
Разобраться, как писать JSX, какие особенности у синтаксиса и как он переводится в вызовы `React.createElement`.

Темы (развернуто)
---
- JSX — это расширение синтаксиса JavaScript, которое выглядит как HTML в JavaScript-коде.
- Встраивание выражений: внутри `{}` можно использовать любые JS-выражения.
- Условный рендеринг: `&&`, тернарный оператор `condition ? A : B`.
- Атрибуты: используйте `className` вместо `class`, `htmlFor` вместо `for`.
- Inline styles: объект в camelCase, например `{ backgroundColor: 'red' }`.
- Fragments позволяют группировать элементы без лишнего DOM-узла: `<>...</>`.

Короткий пример
---

```jsx
function Greeting({ name, isLoggedIn }) {
	return (
		<>
			<h2>Hello {name}</h2>
			{isLoggedIn ? <p>Welcome back!</p> : <p>Please sign in.</p>}
		</>
	);
}
```

Практические рекомендации
---
- Не пишите сложную логику внутри JSX — выносите в переменные или функции.
- Для длинных inline-стилей используйте CSS-модули или классы.
