
# Блок 5 — Списки и ключи (Lists and Keys)

Цель
---
Освоить рендеринг динамических списков в React и понять, зачем нужен атрибут `key`.

Темы
---
- Использование `.map()` для преобразования массивов в список элементов.
- Атрибут `key` помогает React отслеживать элементы между рендерами и минимизировать DOM-операции.
- Предпочтительно использовать устойчивый `id` вместо индекса массива.
- Работа с вложенными списками и производными (фильтрация, сортировка).

Пример
---

```jsx
function UsersList({ users }) {
	return (
		<ul>
			{users.map(u => (
				<li key={u.id}>{u.name}</li>
			))}
		</ul>
	);
}
```
