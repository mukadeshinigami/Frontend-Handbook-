
# Практика — Блок 6: Component Composition

Задачи:
- Создайте layout с Header, Footer и Sidebar.
- Card компонент который принимает `children`.
- Реализуйте Modal и используйте её в примере.
- Tabs или Accordion компонент.

Подсказки:
- Для модального окна используйте портал (`createPortal`) или прикрепление в конец `body`.
- Для Tabs храните индекс активной вкладки в `useState`.
