# 02 — Типы и переменные

Коротко: значение и ссылочные типы, nullable, var, константы, преобразования.

Примеры:

```csharp
int i = 10;
double d = 3.14;
string s = "text";
object o = s; // ссылочный

int? maybe = null; // nullable
var inferred = 5; // int
const double PI = 3.1415;
```

Особенности:
- Boxing/unboxing
- Nullable<T> и оператор ??

Упражнение:
- Напишите функцию, которая безопасно парсит число из строки и возвращает значение по умолчанию, если парсинг не удался.
